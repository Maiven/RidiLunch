# RidiLunch

점심 메뉴를 정해줍니다.

_(최근 3개의 메뉴는 선택되지 않습니다.)_

# How To

RidiLunch PyPI 패키지를 설치하여 사용할 수 있습니다.

```
$ pip install rlunch
$ rlunch
> 아비꼬
```

# Options

- _-d_ : 거리별 추첨 (f: 먼, m: 보통, n: 가까운)
- _-s_ : 종류별 추첨 (k: 한식, j: 일식, c: 중식, a: 양식)
```
$ rlunch -df
> 고슴도치

$ rlunch -sc
> 취홍
```
